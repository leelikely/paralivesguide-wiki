# ParalivesGuide.wiki

A fan-made guide wiki for **Paralives** — the upcoming life simulation game by Paralives Studio, launching **May 25, 2026**.

## 🎮 What is this?

A community-driven resource covering:
- Character creation guides
- Build mode tutorials
- Career & skill breakdowns
- Relationship mechanics
- Modding & custom content

## 🛠 Tech Stack

- HTML + Tailwind CSS (CDN)
- Markdown-based articles
- Hosted on Cloudflare Pages
- Deployed via GitHub

## 📝 Contributing

This is a personal project. If you'd like to suggest content, open an issue!
