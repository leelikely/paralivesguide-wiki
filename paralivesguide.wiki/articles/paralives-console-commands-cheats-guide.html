Paralives Console Commands & Cheats Guide (2026)

Every cheat and console command available in the May 2026 early access version — plus how to enable the console, use commands safely, and fix common issues.

📅 Published: May 22, 2026·🔄 Updated: May 22, 2026
💡 Important: Using console commands disables Steam achievements for the current save and may corrupt your save. Always back up your saves first! Save location: %USERPROFILE%\AppData\LocalLow\Alex Masse\Paralives\Saves

🖥️ How to Enable the Console

  • Windows: Ctrl + Shift + C
  • Mac: Cmd + Shift + C
  • Close: Press Esc or the same keys again.

💰 1. Money & Needs Commands

CommandEffectExample
add_money [amount]Adds Paradimes to the active householdadd_money 100000 (grants 100,000 Paradimes)
fill_needs_allInstantly fills all needs of the selected Para—
freeze_needsToggles freezing of need decay (enter again to disable)—

🏗️ 2. Build Mode Commands

CommandEffect
ignore_placement_rulesDisables placement restrictions: allow overlapping, wall-clipping, and grid-free placement. Enter again to disable.
unlock_all_furnitureUnlocks all career-reward items and hidden furniture.

🕒 3. Time & Environment Commands

CommandEffectExample
set_time [HH:MM]Instantly changes the in‑game time (24‑hour format)set_time 14:00 (sets time to 2 PM)

⚡ 4. Skills & Career Commands

CommandEffectExample
set_skill_level [skill] [level]Sets a specific skill to level 1–10set_skill_level cooking 10
promote_careerInstantly promotes your Para in their current career—

🧬 5. Personality Trait Commands

Format: add_trait [trait_id]

Trait IDEffect
trait_charismaticBoosted relationship gain in conversations.
trait_geniusLogic and programming skills improve much faster.
trait_nightowlEnergy decays slower at night.
trait_neatPara autonomously cleans more often.

🛠️ Troubleshooting

  • Console won't open? Disable overlays (Discord, GeForce Experience) and switch to an English keyboard layout.
  • Command not working? Commands are case‑sensitive (all lowercase) and must include underscores exactly.
  • Disable a cheat: For toggle cheats like freeze_needs, enter the same command again.
