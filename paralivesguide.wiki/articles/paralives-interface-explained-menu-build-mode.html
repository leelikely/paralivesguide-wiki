Paralives Interface Explained: Main Menu to Build Mode

Feeling overwhelmed by all the icons? This guide walks you through every menu, panel, and tool — from the main menu to advanced build mode.

📅 Published: May 22, 2026·🔄 Updated: May 22, 2026

🏠 Main Menu

When you launch the game, you'll see a clean, minimalist menu:

  • ⏵ Continue — Jump into your latest save.
  • 🆕 New Game — Start a fresh world.
  • 📂 Load Game — Pick from previous saves.
  • ⚙️ Settings — Adjust graphics, audio, gameplay, and more.
  • 🧩 Workshop — Browse and manage mods and custom content from the community.
  • 🚪 Exit — Quit to desktop.

🧬 Paramaker (Character Creator)

Paramaker is divided into three main zones:

Left Navigation

  • 👤 Appearance — Face, body proportions, hair, skin details.
  • 👗 Clothing — Outfits for different occasions, supports layering.
  • 🧠 Personality — Traits, talents, social preferences, lifestyle.

Center Preview

  • Click & drag to rotate 360°.
  • Click on any body part to jump to its editor.
  • Top buttons switch between front/side/back views.

Right Toolbar

  • 🎨 Color Wheel — Unlimited colors, hex code support.
  • 🎲 Random — Randomize appearance.
  • 💾 Save/Load — Store your created Paras for later.
  • ✅ Confirm — Finish and enter live mode.

🖥️ Live Mode HUD

Bottom Left

  • 🧑 Portrait — Currently selected Para.
  • 📊 Needs — Hunger, Energy, Hygiene, Social, Fun, Bladder.
  • 😄 Emotion — Current mood (happy, sad, angry, etc.).

Bottom Right

  • 🔨 Build Mode — Enter build mode.
  • 🗺️ Map — Open world map.
  • 📱 Phone — Call, text, check schedules.
  • ⚙️ Settings — In-game settings.

Top Bar

  • 💰 Household Funds — Current Paradimes.
  • 🕒 Time & Date — In-game clock.
  • ⏩ Speed Controls — Normal, Fast, Ultra Fast.
  • ⏸️ Pause — Freeze time.

🔨 Build Mode Interface

Build mode transforms the UI. Here's the breakdown:

Top Left — Core Tools

  • 🧱 Build Tools — Walls, doors, windows, stairs, roofs.
  • 🛏️ Room Tools — Furniture categorized by room.
  • 🪑 Furniture Tools — Individual furniture types.
  • 🌳 Terrain Tools — Sculpt terrain, plants, rocks.
  • 🔍 Search Box — Find any object instantly.

Top Right — Utility Bar

  • 💰 Funds — Budget remaining.
  • 🖱️ Cursor Mode — Select / Move.
  • 🔨 Hammer — Delete objects (S).
  • 💉 Eyedropper — Copy style (P).
  • 🚜 Bulldozer — Clear lot.
  • ↩️ Undo / Redo — Step back or forward.
  • 🏢 Floor Toggle — Switch between floors.
  • 👁️ Wall Visibility — Hide walls to see inside.
  • 🌗 Time of Day — Preview lighting.
  • 📸 Photo Mode — Capture your build.

Bottom — Object Editor

When you select an item, a bar appears allowing you to:

  • 🔲 Resize
  • ↻ Rotate
  • 🎨 Change color / material
  • 📋 Duplicate
  • 🗑️ Delete

🛒 Buy Mode

Buy mode activates when you click "Room Tools" or "Furniture Tools". Items are sorted by category and price. Click to preview in 3D, then place them inside your house.

The interface may seem complex, but after a few hours it becomes second nature. Start with a small house to practice!
